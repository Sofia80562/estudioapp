# Spec: 012 · Gestión de Materias

## Estado
⏳ **Propuesta**

## Qué hace
Gestión centralizada de las materias (Subjects) asociadas a una organización o carrera, permitiendo estructurar el contenido académico y vincularlo con tareas y docentes.

## Criterios de aceptación
- [ ] CRUD de materias con validación de código de materia único.
- [ ] Relación con `Organization` y `Venue`.
- [ ] Asignación de docentes a materias.
- [ ] Documentación Swagger completa.
