# Spec: 015 · Panel de Reportes y Analytics

## Estado
⏳ **Propuesta**

## Qué hace
Dashboard administrativo para visualizar métricas de uso, rendimiento académico promedio por materia y actividad de los usuarios dentro de `estudioapp`.

## Criterios de aceptación
- [ ] Endpoints de solo lectura para agregación de datos (KPIs).
- [ ] Reportes exportables en formato PDF (integración con la feature de reportes).
- [ ] Filtrado avanzado por rango de fechas y materia.
