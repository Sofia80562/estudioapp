# Spec: 013 · Gestión de Calificaciones y Progreso

## Estado
⏳ **Propuesta**

## Qué hace
Módulo para registrar, calcular y visualizar el rendimiento académico del estudiante en cada materia, permitiendo tanto la calificación manual por parte del docente como el cálculo automático basado en tareas.

## Criterios de aceptación
- [ ] CRUD de calificaciones vinculadas a `Task` o `Subject`.
- [ ] Registro de escalas de calificación configurables por organización.
- [ ] Vista de progreso académico (promedio ponderado).
- [ ] Documentación Swagger completa.
