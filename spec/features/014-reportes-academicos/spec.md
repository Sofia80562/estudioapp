# Spec: 014 · Notificaciones y Avisos

## Estado
⏳ **Propuesta**

## Qué hace
Sistema de notificaciones internas para mantener a los usuarios informados sobre fechas de entrega próximas, nuevas calificaciones o anuncios de los docentes.

## Criterios de aceptación
- [ ] Sistema de eventos para disparar notificaciones (Trigger-based).
- [ ] CRUD de notificaciones para el usuario.
- [ ] Estado de "leído/no leído".
- [ ] Endpoint para marcado masivo como leído.
